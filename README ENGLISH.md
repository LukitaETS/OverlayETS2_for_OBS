#  **Installation Tutorial – ETS2 Overlay**

##  Download

First, download the file from MediaFire:
https://www.mediafire.com/file/ma3r7pa9tpjsork/All_Here.zip/file

##  Step 1: Extract the main file

Once the file is downloaded, right-click it and select **“Extract here”** or **“Extract to…”**. This will create a folder with all the contents.

##  Step 2: Extract internal folders

Inside the folder, you will find **Overlay V1** and **ETS2 Telemetry Server**. You must extract both folders as well, using the same method (right-click → extract).

##  Step 3: Copy the overlay

Open the **Overlay V1** folder and copy all the files inside.

##  Step 4: Go to the correct path

Now open the **ETS2 Telemetry Server** folder and follow this path:
`ets2-telemetry-server-master > server > Html > skins > template`
The **skins** folder contains all the overlays included with the program, and in this case we will use **template** because it is the most basic one and safe to replace.

##  Step 5: Replace files

Open the **template** folder and paste the files you copied earlier. When prompted, select **replace all files**.

##  Step 6: Run the server

Go back until you find **Ets2Telemetry.exe** and open it.

##  Step 7: Activate the overlay

Inside the program, look for **“HTML5 App URL”**, scroll down until you find **“Dashboard Skin Template”**, and click on it to activate the overlay.

##  Step 8: Launch the game

Open **Euro Truck Simulator 2** and the overlay should now be working correctly.

##  Notes

Make sure everything is properly extracted, the files were correctly replaced, and avoid modifying other folders inside **skins** unless you know what you are doing.
