# 📥 **Tutorial de instalación – Overlay ETS2**      

## 🔗 Descarga

Primero descargá el archivo desde MediaFire:
https://www.mediafire.com/file/ma3r7pa9tpjsork/All_Here.zip/file

## 📦 Paso 1: Descomprimir archivo principal

Una vez descargado el archivo, hacé clic derecho y seleccioná **“Extraer aquí”** o **“Extraer en…”**. Esto va a generar una carpeta con todo el contenido.

## 📂 Paso 2: Descomprimir carpetas internas

Dentro de la carpeta vas a encontrar **Overlay V1** y **ETS2 Telemetry Server**. Tenés que descomprimir las dos carpetas también, haciendo lo mismo que antes (clic derecho → extraer).

## 🎨 Paso 3: Copiar el overlay

Entrá en la carpeta **Overlay V1** y copiá absolutamente todos los archivos que hay dentro.

## 📁 Paso 4: Ir a la ruta correcta

Ahora abrí la carpeta de **ETS2 Telemetry Server** y seguí esta ruta:
`ets2-telemetry-server-master > server > Html > skins > template`
La carpeta **skins** contiene todos los overlays que vienen con el programa, y en este caso vamos a usar **template** porque es el más básico y no afecta reemplazarlo.

## 🔄 Paso 5: Reemplazar archivos

Entrá en la carpeta **template** y pegá todos los archivos que copiaste antes. Cuando Windows te pregunte, seleccioná **reemplazar todo**.

## ▶️ Paso 6: Ejecutar el servidor

Volvé hacia atrás hasta encontrar el archivo **Ets2Telemetry.exe** y abrilo.

## ⚙️ Paso 7: Activar el overlay

Dentro del programa, buscá la opción **“HTML5 App URL”**, bajá hasta encontrar **“Dashboard Skin Template”** y hacé clic ahí para activar el overlay.

## 🚛 Paso 8: Abrir el juego

Abrí **Euro Truck Simulator 2** y el overlay ya debería estar funcionando correctamente.

## ⚠️ Notas

Asegurate de haber descomprimido todo correctamente, de haber reemplazado bien los archivos y de no modificar otras carpetas dentro de **skins** si no sabés lo que estás haciendo.
