If you already have the game open and the overlay running, copy the link provided by the Telemetry Server page. Then open OBS, go to **Add Source**, and select **“Browser”**. Give it any name you want and click accept.

In the **URL** field, paste the link you just copied, click accept, and adjust the size and position to fit your scene.

Remember that the overlay is fully **open source**, so if you want to make it more transparent or tweak small details, you are completely free to do so.
